# Distribuição do Número de Candidatos do ENEM

Aplicação desenvolvida para a disciplina de Banco de Dados Geográficos, semestre 2023/2, **Trabalho Prático 01 - Mapa Temático**, tendo como objetivo a exibição da distribuição do número de candidatos do ENEM (Exame Nacional do Ensino Médio) por cidade com o decorrer dos anos dentro do período de 2013 a 2022.

> Acesso à aplicação: [Candidatos ENEM](https://candidatos-enem.streamlit.app/)

Ferramentas utilizadas
> Hospedagem da aplicação: [Streamlit](https://streamlit.io/) </br>Construção do mapa: [Plotly](https://plotly.com/python/maps/) </br>Serviço de localização: [Mapbox](https://www.mapbox.com/) </br>Fonte dos dados: [Microdados Abertos do ENEM](https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/microdados/enem)
